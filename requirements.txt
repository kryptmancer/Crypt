# Bletchley Baudot Cryptanalysis Suite - Requirements

## Python Version
Python 3.6 or higher

## Standard Library Dependencies
All dependencies are part of Python's standard library:
- tkinter (for GUI - usually pre-installed with Python)
- sys
- os

## No External Packages Required
This project uses only Python standard library modules.

## Platform Support
- macOS ✓
- Linux ✓
- Windows ✓ (requires Python with tkinter)

## Installation Verification

To verify your Python installation supports this tool:

```bash
# Check Python version
python3 --version

# Check if tkinter is available (for GUI)
python3 -c "import tkinter; print('Tkinter available')"
```

## Optional: Create Virtual Environment

While not required (no external dependencies), you can use a virtual environment:

```bash
python3 -m venv venv
source venv/bin/activate  # On macOS/Linux
# venv\Scripts\activate    # On Windows
```

## Quick Start

```bash
# No installation needed! Just run:
python3 BaudotSolver.py    # CLI version
python3 SolverUI.py        # GUI version
python3 test_suite.py      # Run tests
```
