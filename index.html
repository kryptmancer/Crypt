The Mathematics

When two messages P₁ and P₂ are encrypted with the same key K:

C₁ = P₁ ⊕ K
C₂ = P₂ ⊕ K

XORing the ciphertexts cancels out the key:

C₁ ⊕ C₂ = P₁ ⊕ P₂

How It Works

  1. Enter two hex ciphertexts encrypted with the same key
  2. Compute the XOR stream (C₁ ⊕ C₂ = P₁ ⊕ P₂)
  3. Try common cribs: THE, ATTACK, REPORT, SECRET
  4. Look for readable results in the output
  5. Build incrementally using discovered fragments

Verification

Implements Bletchley Park Baudot encoding:

K (11110) ⊕ 5 (11011) = H (00101) ✓
